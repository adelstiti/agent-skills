# Agent Skills Repository

This repository contains comprehensive skills for AI coding agents, optimized for:
- Laravel 12 + Lucid Architecture
- Next.js 15 (App Router)
- TypeScript
- React 19
- Tailwind CSS

## Skills Overview

| Skill | Rules | Priority Focus |
|-------|-------|----------------|
| laravel-lucid-architecture | 14 | Lucid domains, features, operations, jobs |
| nextjs-best-practices | 11 | App Router, RSC, caching, performance |

## Usage

Skills in `skills/` directory follow the Agent Skills format. Each skill has:
- `SKILL.md` - Main instructions with YAML frontmatter
- `rules/` - Detailed rules (loaded on demand)

## Contributing

When adding new rules:
1. Follow the existing rule template in `rules/_template.md`
2. Include incorrect and correct code examples
3. Explain the "why" not just the "what"
4. Add appropriate priority level
